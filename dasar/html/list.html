Unordered list (Daftar Tidak Terurut)

Contoh HTML untuk unordered list:

  • Item 1
  • Item 2
  • Item 3

Secara default simbol lingkaran unordered list dapat digantikan dengan atribut type.

berikut yang bisa diberikan untuk nilai atribut type:

  • disc (default) daftar dengan tanda penomoran berbentuk lingkaran
  • square daftar dengan tanda penomoran berbentuk persegi
  • circle daftar dengan tanda penomoran berbentuk lingkaran yang lebih besar daripada "disc"
  • none daftar tanpa tanda penomoran atau bullets

Silahkan lihat pada contoh sederhana dibawah:

Unordered list Disc

  • Item 1
  • Item 2
  • Item 3

Unordered list square

  • Item A
  • Item B
  • Item C

Unordered list circle

  • Item A
  • Item B
  • Item C

Unorderes none

  • Item X
  • Item Y
  • Item Z
Selain menggunakan atribut type seperti contoh diatas, kita dapat menggunakan gambar style gambar yang dilakukan dengan menggunakan style css.

Ordered List (Daftar Terurut)

  2. Item 2
  3. Item 3

untuk membuat seperti a, b, c atau angka romawi seperti I, II, III. Diperlukan untuk menggunakan atribut type yang diletakkan pada tag ol

Description List di HTML

Description List dalam HTML digunakan untuk menyajikan definisi atau deskripsi dari sebuah kata kunci atau istilah. Description List terdiri dari dua bagian: istilah (term) dan deskripsi (description). Istilah ditempatkan dalam elemen <dt> (Definition Term), sedangkan deskripsi ditempatkan dalam elemen <dd> (Definition Description).

Terkait dengan Description List di HTML, ada tiga elemen HTML yang digunakan:

  1. <dl> (Definition List): Elemen <dl> digunakan untuk membungkus seluruh Description List. Itu adalah elemen utama yang mengelilingi semua pasangan istilah dan deskripsi dalam list.

  2. <dt> (Definition Term): Elemen <dt> digunakan untuk mendefinisikan istilah atau kata kunci yang akan dijelaskan atau didefinisikan dalam list. Ini biasanya ditempatkan diikuti oleh elemen <dd>.

  3. <dd> (Definition Description): Elemen <dd> digunakan untuk memberikan deskripsi atau definisi dari istilah yang didefinisikan dalam elemen <dt>. Elemen <dd> biasanya mengikuti elemen <dt> dan berisi penjelasan atau definisi tersebut.
