URL in HTML page index 
    URL singkat untuk "Uniform Resource Locator." URL adalah alat penting dalam menghubungkan dan mengidentifikasi sumber daya di web.
    Terdapat dua jenis URL utama dalam konteks HTML: URL absolut dan URL relatif.
    Di sini, saya akan memberikan contoh URL relatif dan beberapa jenisnya:

URL relatif adalah URL yang merujuk ke sumber daya dengan mengacu pada lokasi relatif terhadap sumber daya yang sedang diakses.
Ini berguna ketika Anda ingin merujuk ke file atau halaman lain dalam direktori yang sama atau direktori yang terkait.

    Referensi ke file dalam direktori yang sama:
        File yang sedang diakses: http://contoh.com/direktori/halaman.html
        URL relatif untuk gambar di direktori yang sama: gambar.jpg

    Referensi ke file dalam direktori yang berbeda:
        File yang sedang diakses: http://contoh.com/direktori/halaman.html
        URL relatif untuk gambar di direktori yang berbeda: ../gambar.jpg

    Referensi ke file dalam subdirektori:
        File yang sedang diakses: http://contoh.com/direktori/halaman.html
        URL relatif untuk gambar dalam subdirektori: subdirektori/gambar.jpg

    Referensi ke akar situs web:
        File yang sedang diakses: http://contoh.com/direktori/halaman.html
        URL relatif ke halaman beranda: /index.html

    Referensi ke parent directory:
        File yang sedang diakses: http://contoh.com/direktori/subdirektori/halaman.html
        URL relatif untuk kembali ke direktori utama: ../../index.html

URL relatif sangat berguna ketika Anda ingin membuat situs web yang dapat dipindahkan dengan mudah atau jika Anda ingin menghindari penggunaan URL absolut yang panjang.
Anda hanya perlu merujuk ke sumber daya dengan cara yang relatif terhadap halaman saat ini.

Namun, perlu diingat bahwa penggunaan URL relatif harus dipertimbangkan dengan cermat untuk memastikan bahwa sumber daya yang diacu dapat ditemukan dengan benar pada semua halaman Anda.
